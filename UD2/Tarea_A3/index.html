Descripción de los siguientes términos:

  1. W3C

    World Wide Web Consortium, organización que desarrolla estándares web.

  2. WHATWG

    Web Hypertext Application Technology Working Group, un grupo que colabora en la especificación de HTML.

  3. HTML

    Lenguaje de marcado para la creación de páginas web.

  4. XHTML

    Extensible HyperText Markup Language, una versión más estricta y compatible con XML de HTML.

  5. DTD

    Document Type Definition, define la estructura y las reglas de un documento XML o HTML.

  6. APIs:
    1. Drag and Drop

      API que permite arrastrar y soltar elementos en una página web.

    2. Geolocalización

      API que permite obtener la ubicación geográfica del usuario.

    3. Almacenamiento local

      API que permite almacenar datos en el navegador del usuario de forma local.

    4. App Cache

      API que permite que las aplicaciones web se almacenen en caché para uso offline.

    5. WebWorkers

      API que permite ejecutar scripts en segundo plano para mejorar el rendimiento.

  7. Editor WYSIWYG

    Editor de texto que muestra el contenido de la misma forma en que aparecerá en el navegador (What You See Is What You Get).

  8. Navegadores:
    1. Mozilla Firefox

      Navegador web de código abierto desarrollado por Mozilla Foundation.

    2. Google Chrome

      Navegador web desarrollado por Google, conocido por su velocidad y soporte de estándares.

    3. Safari

      Navegador web desarrollado por Apple, principalmente para dispositivos Apple.

    4. Opera

      Navegador web conocido por su uso de tecnologías innovadoras y su bajo consumo de recursos.
